# Global Sales Analysis (2010–2017)

This project explores a global sales dataset to understand performance across 
product categories, regions, and sales channels. The notebook includes data 
cleaning, exploratory analysis, feature engineering, financial metrics, 
visualizations, and business insights.

## Objectives
- Clean and prepare the dataset  
- Perform exploratory data analysis  
- Evaluate financial KPIs and profitability  
- Analyze performance by region, product category, and sales channel  
- Study temporal trends and shipping-time dynamics  

## Repository Structure
- `notebook/` – Jupyter notebook with full analysis  
- `figs/` – Exported visualizations  
- `data/` – (Not included) Instructions for dataset location  
- `README.md` – Project documentation  

## Key Findings
- Europe represents the primary market, generating 89% of total profit  
- Cosmetics is the most profitable category  
- Online and Offline channels show similar performance  
- No correlation found between shipping time and order profitability  
- Revenue and profit peak in 2012, followed by a steady decline  

## Tools and Libraries
- Python, Pandas, NumPy  
- Matplotlib, Seaborn  
- Jupyter Notebook
