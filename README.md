# 📊 Global Sales Analysis (2010–2017)
### Python · Pandas · Matplotlib · Seaborn · Google Colab

An exploratory data analysis project focused on understanding global sales 
performance across product categories, regions, and sales channels. Includes 
data cleaning, feature engineering, financial metrics, and visual insights.

---

| KPI | Value |
|-----|-------|
| Total Orders | 1,328 |

| Total Profit | 501M USD |
| Profit Margin | 34.1% |
| Countries Covered | 45 |
| Period | 2010–2017 |

---

## 🔍 Main Insights

- **Cosmetics** is the most profitable category.
- **Europe** generates ~89% of total profit.
- **Online** and **Offline** channels perform similarly.
- **Shipping time** does *not* correlate with profitability.
- Revenue and profit peak in **2012**, then decline.
- Orders are evenly distributed across weekdays.
