# 📊 Global Sales Analysis (2010–2017) — Python

Exploratory data analysis project focused on global sales performance across product categories, regions and sales channels. Includes data cleaning, feature engineering, financial metrics and visual insights.

---

## ✅ Key Metrics

| KPI | Value |
|---|---|
| Total Orders | 1,328 |
| Total Revenue | $1,702M USD |
| Total Profit | $501M USD |
| Profit Margin | 34.1% |
| Countries Covered | 45 |
| Period | 2010–2017 |

---

## 🔍 Main Insights

- **Cosmetics** is the most profitable category despite not leading in revenue.
- **Europe** generates ~89% of total profit.
- **Online and Offline** channels perform similarly across all categories.
- **Shipping time** does not correlate with profitability.
- Revenue and profit peak in **2012**, then steadily decline.
